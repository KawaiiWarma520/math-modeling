层次分析法（Analytic Hierarchy Process, AHP）是由美国运筹学家 Saaty 教授于 20 世纪 70 年代提出的一种**多目标决策分析方法**。它特别适合那些**难以完全定量分析**的问题，在数学建模比赛中出镜率非常高。

## 核心思想

AHP 的核心就八个字：**先分解，后综合**。

1. 把复杂问题分解为若干层次和因素
2. 通过两两比较确定各因素的相对重要性
3. 最终得出各方案的权重排序

## 基本步骤

### 第一步：建立层次结构模型

一般分为三层：

- **目标层**：要解决的问题
- **准则层**：评价的维度/指标
- **方案层**：备选方案

### 第二步：构造判断矩阵

这是 AHP 最关键的一步。比如我们要比较三个准则 A、B、C 的重要性：

|   | A | B | C |
|---|---|---|---|
| A | 1 | 3 | 5 |
| B | 1/3 | 1 | 2 |
| C | 1/5 | 1/2 | 1 |

其中数字含义：
- `1` = 同样重要
- `3` = 稍微重要
- `5` = 明显重要
- `7` = 强烈重要
- `9` = 极端重要

> 偶数 2、4、6、8 表示相邻判断的中间值

### 第三步：计算权重向量

常用方法：**几何平均法**或**特征值法**。

### 第四步：一致性检验

因为人的判断不一定完全一致，需要用 **CR（一致性比率）** 来检验：

$$
CR = \frac{CI}{RI}
$$

- $CI = \frac{\lambda_{max} - n}{n-1}$
- $RI$ 查表可得
- 若 $CR < 0.1$，则判断矩阵一致性可以接受

## MATLAB 代码示例

```matlab
function [w, CR] = AHP(A)
    % A: 判断矩阵
    % w: 权重向量
    % CR: 一致性比率
    
    [V, D] = eig(A);
    lambda_max = max(diag(D));
    n = size(A, 1);
    
    CI = (lambda_max - n) / (n - 1);
    RI = [0 0 0.52 0.89 1.12 1.26 1.36 1.41 1.45 1.49];
    CR = CI / RI(n);
    
    % 几何平均法求权重
    w = prod(A, 2) .^ (1/n);
    w = w / sum(w);
    
    if CR < 0.1
        disp('一致性检验通过！');
    else
        disp('一致性检验不通过，请调整判断矩阵');
    end
end
```

## 实战 tips

1. **判断矩阵别太大**：n ≤ 9 为宜，不然一致性很难通过
2. **方案层要具体**：方案之间要有可区分度
3. **可以结合其它方法**：AHP 定权重 + TOPSIS 排序，是经典组合
