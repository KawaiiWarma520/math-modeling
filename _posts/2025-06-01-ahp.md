## 🧠 基本概念

**一句话定义**：层次分析法（AHP）是一种把模糊的"主观判断"变成可量化的"数学权重"的多准则决策方法。

**核心思想**：将复杂决策问题拆解成"目标-准则-方案"三层结构，通过两两比较的方式确定各因素的重要性权重，最终加权得出方案排序。

**三个层次**：
- **目标层**：要解决的核心问题（如：选最好的手机）
- **准则层**：评价维度（如：价格、性能、续航）
- **方案层**：备选方案（如：手机A、手机B、手机C）

## 📊 适用场景

AHP 特别适合：
- 缺乏历史数据的主观评价问题
- 多准则决策问题（选方案、评优劣）
- 结合其他模型使用：AHP定权重 + TOPSIS排序（经典组合）

## 🛠️ 解题步骤

**Step 1：建立层次结构**

画出一个三层结构：目标 → 准则 → 方案

**Step 2：构造判断矩阵**

对每个层级的元素进行两两比较，用 1-9 标度法：

| 标度 | 含义 |
| :--- | :--- |
| 1 | 同样重要 |
| 3 | 稍微重要 |
| 5 | 明显重要 |
| 7 | 强烈重要 |
| 9 | 极端重要 |
| 2,4,6,8 | 相邻判断的中间值 |

判断矩阵示例：

| | 价格 | 性能 | 续航 |
| :--- | :--- | :--- | :--- |
| 价格 | 1 | 1/3 | 2 |
| 性能 | 3 | 1 | 4 |
| 续航 | 1/2 | 1/4 | 1 |

**Step 3：计算权重**

常用**特征值法**或**几何平均法**。

**Step 4：一致性检验**

不能出现"A>B, B>C, 但C>A"这种矛盾。用 CR（一致性比率）检验：

$$CR = \frac{CI}{RI}$$

若 $CR < 0.1$，一致性可接受。

## ⚠️ 注意事项

- **判断矩阵不要太大**（n ≤ 9），否则一致性很难通过
- **主观性强**，建议结合专家意见或群体决策
- **可以结合熵权法**做客观赋权，形成主客观组合权重

## 💻 代码实现

```matlab
function [w, CR] = AHP(A)
    % A: 判断矩阵
    % w: 权重向量
    % CR: 一致性比率

    [V, D] = eig(A);
    lambda_max = max(diag(D));
    n = size(A, 1);

    CI = (lambda_max - n) / (n - 1);
    RI = [0 0 0.52 0.89 1.12 1.26 1.36 1.41 1.45 1.49];
    CR = CI / RI(n);

    % 几何平均法求权重
    w = prod(A, 2) .^ (1/n);
    w = w / sum(w);

    if CR < 0.1
        disp('一致性检验通过！');
    else
        disp('一致性检验不通过，请调整判断矩阵');
    end
end
```
