## 🧠 基本概念

**一句话定义**：层次分析法（AHP）是一种把模糊的"主观判断"变成可量化的"数学权重"的多准则决策方法。

**核心思想**：将复杂决策问题拆解成"目标-准则-方案"三层结构，通过两两比较的方式确定各因素的重要性权重，最终加权得出方案排序。

**三个层次**：
- **目标层**：要解决的核心问题（如：选最好的手机）
- **准则层**：评价方案所用到的各个维度（如：价格、性能、续航）
- **方案层**：待选的具体方案（如：iPhone、华为、小米）

## 📊 适用场景

| 场景 | 问题类型 | 例子 |
|:---|:---|:---|
| 方案选择 | 从多个选项中选最优 | 选择供应商、选址、选旅游地 |
| 权重确定 | 给多个指标分配权重 | 幸福指数各维度权重、学生评价指标 |
| 排序问题 | 对多个对象进行排名 | 城市宜居度排名、员工绩效排名 |
| 评价体系构建 | 建立综合评价模型 | 空气质量评价、水质评价 |

**比赛中常见应用**：
- 美赛 E 题（环境类）：评价不同政策的综合效果
- 美赛 F 题（政策类）：给不同方案打分排序
- 国赛评价类题目：确定各评价指标的权重

## 🛠️ 解题步骤

**Step 1：建立层次结构图**
- 画出目标层 → 准则层 → 方案层的树状图
- 用 draw.io 或 AxGlyph 绘制，放入论文

**Step 2：构造准则层的判断矩阵**
- 使用 1-9 标度法，两两比较各准则的重要性
- 矩阵元素 $a_{ij}$ 表示：第 i 个因素比第 j 个因素重要多少

**Step 3：计算权重（方根法）**
- 每行元素相乘 → 开 n 次方 → 归一化

**Step 4：一致性检验**
- 计算 CR 值，判断矩阵是否逻辑一致
- CR < 0.1 才可接受

**Step 5：构造方案层的判断矩阵**
- 对每个准则，重复 Step 2-4，得到各方案在该准则下的得分

**Step 6：计算方案总得分**
- 总得分 = Σ(准则权重 × 方案在该准则下的得分)
- 得分最高的方案即为最优

## ⚠️ 注意事项

**1. 判断矩阵填写规则**

> **行比列，重要就大于 1，不如就小于 1，对角线上都是 1，对称位置互为倒数**

| 比较结果 | $a_{ij}$ 的值 | $a_{ji}$ 的值 |
|:---|:---|:---|
| i 比 j 同等重要 | 1 | 1 |
| i 比 j 稍微重要 | 3 | 1/3 |
| i 比 j 明显重要 | 5 | 1/5 |
| i 比 j 强烈重要 | 7 | 1/7 |
| i 比 j 极端重要 | 9 | 1/9 |
| 介于上述之间 | 2,4,6,8 | 1/2,1/4,1/6,1/8 |

**2. 必须做一致性检验**
- 很多队伍只算权重不做检验，评委一眼就能看出来
- CR < 0.1 必须写在论文里

**3. 判断矩阵的来源要交代清楚**
- 写明："判断矩阵基于专家打分/文献调研/团队讨论确定"
- 附上完整的判断矩阵

**4. 层次结构图一定要画**
- 评委喜欢看图，清晰的三层结构是加分项

**5. AHP 的局限性**
- 主观性强，依赖判断矩阵质量
- 层次太多时计算量大
- 新方案加入时需要重新计算

## ✏️ 论文写法

**摘要中的表述**：
> 本文采用层次分析法（AHP）对...进行综合评价。首先建立层次结构模型，构造判断矩阵，计算各指标权重并完成一致性检验（CR < 0.1），最终得到各方案的排序结果为...

**模型建立部分的模板**：

> **1. 建立层次结构模型**
>
> 将问题分为三层：目标层 A、准则层 B（共 m 个指标）、方案层 C（共 n 个方案）。
>
> **2. 构造判断矩阵**
>
> 采用 1-9 标度法，对同一层次的各因素进行两两比较，构造判断矩阵 $A = (a_{ij})_{m \times m}$，其中 $a_{ij}$ 表示因素 i 对因素 j 的相对重要性，满足 $a_{ij} > 0$，$a_{ij} = 1/a_{ji}$，$a_{ii} = 1$。
>
> 准则层判断矩阵如下：
>
> $$
> A = \begin{bmatrix}
> 1 & 1/3 & 3 \\
> 3 & 1 & 5 \\
> 1/3 & 1/5 & 1
> \end{bmatrix}
> $$
>
> **3. 计算权重向量**
>
> 采用方根法计算权重：
> - 计算每行元素的乘积：$M_i = \prod_{j=1}^{m} a_{ij}$
> - 计算 $M_i$ 的 m 次方根：$\overline{w_i} = \sqrt[m]{M_i}$
> - 归一化：$w_i = \overline{w_i} / \sum_{j=1}^{m} \overline{w_j}$
>
> **4. 一致性检验**
>
> - 计算最大特征根 $\lambda_{max} = \frac{1}{m} \sum_{i=1}^{m} \frac{(A w)_i}{w_i}$
> - 一致性指标 $CI = \frac{\lambda_{max} - m}{m - 1}$
> - 一致性比率 $CR = CI / RI$
>
> 当 $CR < 0.1$ 时，认为判断矩阵的一致性可以接受。

**结果分析中的表述**：
> 由准则层权重可知，性能（0.637）是最重要的因素，价格（0.258）次之，续航（0.105）影响最小。方案层计算显示，华为的总得分最高（0.55），因此推荐华为手机。

## 💻 代码实现

```python
import numpy as np

def ahp(matrix):
    """计算AHP权重和一致性检验"""
    n = len(matrix)
    
    # Step 1: 方根法计算权重
    # 每行元素相乘
    M = np.prod(matrix, axis=1)
    # 开 n 次方
    w_bar = M ** (1/n)
    # 归一化
    w = w_bar / np.sum(w_bar)
    
    # Step 2: 计算最大特征根 lambda_max
    # 判断矩阵 × 权重向量
    Aw = np.dot(matrix, w)
    # 计算每个元素的比值
    lambda_i = Aw / w
    lambda_max = np.mean(lambda_i)
    
    # Step 3: 一致性检验
    CI = (lambda_max - n) / (n - 1)
    RI_dict = {1:0, 2:0, 3:0.52, 4:0.89, 5:1.12, 6:1.26, 7:1.36, 8:1.41, 9:1.46}
    RI = RI_dict.get(n, 1.46)
    CR = CI / RI
    
    return w, lambda_max, CI, CR

# 准则层判断矩阵（价格、性能、续航）
criteria_matrix = np.array([
    [1, 1/3, 3],
    [3, 1, 5],
    [1/3, 1/5, 1]
])

# 计算准则层权重
w_criteria, lam, CI, CR = ahp(criteria_matrix)
print("=== 准则层 ===")
print(f"权重: 价格={w_criteria[0]:.3f}, 性能={w_criteria[1]:.3f}, 续航={w_criteria[2]:.3f}")
print(f"lambda_max={lam:.4f}, CI={CI:.4f}, CR={CR:.4f}")
print(f"一致性检验: {'通过' if CR < 0.1 else '不通过'}")

# 方案层判断矩阵（价格准则下：iPhone, 华为, 小米）
price_matrix = np.array([
    [1, 1/3, 1/2],
    [3, 1, 2],
    [2, 1/2, 1]
])

# 方案层判断矩阵（性能准则下）
perf_matrix = np.array([
    [1, 1/2, 1/3],
    [2, 1, 1/2],
    [3, 2, 1]
])

# 方案层判断矩阵（续航准则下）
battery_matrix = np.array([
    [1, 2, 3],
    [1/2, 1, 2],
    [1/3, 1/2, 1]
])

# 计算方案层权重
w_price, _, _, _ = ahp(price_matrix)
w_perf, _, _, _ = ahp(perf_matrix)
w_battery, _, _, _ = ahp(battery_matrix)

print("\n=== 方案层 ===")
print(f"价格准则下: iPhone={w_price[0]:.3f}, 华为={w_price[1]:.3f}, 小米={w_price[2]:.3f}")
print(f"性能准则下: iPhone={w_perf[0]:.3f}, 华为={w_perf[1]:.3f}, 小米={w_perf[2]:.3f}")
print(f"续航准则下: iPhone={w_battery[0]:.3f}, 华为={w_battery[1]:.3f}, 小米={w_battery[2]:.3f}")

# 计算总得分
scores = np.array([
    w_criteria[0] * w_price + w_criteria[1] * w_perf + w_criteria[2] * w_battery
])
scores = scores.flatten()

print("\n=== 最终得分 ===")
print(f"iPhone: {scores[0]:.3f}")
print(f"华为: {scores[1]:.3f}")
print(f"小米: {scores[2]:.3f}")
print(f"推荐方案: {['iPhone', '华为', '小米'][np.argmax(scores)]}")
```
