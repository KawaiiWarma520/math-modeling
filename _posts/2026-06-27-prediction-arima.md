# 📖 故事引入

某电商平台想预测未来一个月的商品销量。

他们手里只有一份数据：

- 过去36个月的销量变化
    

但问题是：

- 销量有波动
    
- 有趋势
    
- 还有周期性变化
    

团队尝试用简单方法：

- 平均值 ❌ 太粗糙
    
- 线性回归 ❌ 忽略时间结构
    
- 灰色预测 ❌ 适合小样本但不够稳定
    

于是他们提出一个问题：

> **能不能直接用“历史规律”来预测未来？**

今天要学习的，就是解决这类问题的经典模型——**ARIMA模型**。

---

# 🧠 什么是 ARIMA？

一句话来说：

> **ARIMA，就是通过“过去的值 + 过去的误差 + 差分处理”，来预测未来时间序列的模型。**

它专门解决：

> 有趋势、有波动、有噪声的时间序列问题

---

# 🚩 什么时候想到 ARIMA？

如果题目中出现下面这些情况，就可以考虑 ARIMA：

- 时间序列预测
    
- 有明显趋势或波动
    
- 数据量中等以上
    
- 需要较高精度预测
    
- 灰色预测不够用
    

例如：

- 销量预测
    
- 股票价格（基础建模）
    
- 人流量预测
    
- 需求预测
    

一个简单判断方法：

> **只要是“连续时间数据预测”，优先考虑 ARIMA。**

---

# 🧠 ARIMA 的核心结构

ARIMA由三部分组成：

---

## ① AR（自回归）

用过去的值预测现在：

$$  
x_t = a_1 x_{t-1} + a_2 x_{t-2} + ...  
$$

👉 本质：历史影响现在

---

## ② I（差分）

让非平稳数据变平稳：

$$  
x'_t = x_t - x_{t-1}  
$$

👉 本质：去趋势

---

## ③ MA（滑动平均）

用过去误差修正预测：

$$  
x_t = \epsilon_t + b_1 \epsilon_{t-1}  
$$

👉 本质：修正误差

---

# 🧠 ARIMA整体形式

$$  
ARIMA(p, d, q)  
$$

- p：AR阶数
    
- d：差分次数
    
- q：MA阶数
    

---

# 🏆 一个完整案例

某电商平台记录了12个月销量：

|月份|销量|
|---|--:|
|1|120|
|2|130|
|3|128|
|4|140|
|5|150|
|6|155|
|7|160|
|8|165|
|9|170|
|10|175|
|11|180|
|12|185|

---

## 今天要解决的问题：

> 预测第13个月销量

---

## 建模思路

我们观察：

- 有明显上升趋势
    
- 不是随机波动
    
- 可以建时间序列模型
    

---

## ARIMA思路：

1. 差分 → 去趋势
    
2. 建立AR关系
    
3. 修正误差
    

---

# 💻 Python 实现

```python
import numpy as np
import pandas as pd
from statsmodels.tsa.arima.model import ARIMA

# 销量数据
data = np.array([120,130,128,140,150,155,160,165,170,175,180,185])

# 转换为时间序列
series = pd.Series(data)

# 建立ARIMA模型
model = ARIMA(series, order=(2,1,1))  # (p,d,q)
model_fit = model.fit()

# 预测未来1期
forecast = model_fit.forecast(steps=1)

print("第13个月预测销量：", forecast.values[0])
```

---

# 🧠 ARIMA的核心理解

一句话总结：

> ARIMA = “趋势处理 + 历史依赖 + 误差修正”

# 📌 本章小结

ARIMA解决的是：

> **复杂时间序列的预测问题**

学习ARIMA时，可以记住三个关键词：

1. **AR（依赖历史）**
    
2. **I（稳定序列）**
    
3. **MA（修正误差）**

## 🔥 和其他预测模型的区别

|模型|特点|
|---|---|
|灰色预测|小样本|
|回归模型|静态关系|
|ARIMA|动态时间依赖|

---

> **灰色预测看“趋势”，ARIMA看“时间结构”。**
