# 📖 故事引入

城市规划部门正在评估几个城市的发展质量。

他们希望从多个维度进行比较，比如：

- 经济水平；
    
- 环境质量；
    
- 交通便利程度；
    
- 教育资源。
    

问题来了：

如果让专家来打分，不同专家的判断可能完全不同；如果领导来定权重，又容易带有主观偏好。

于是一个更现实的问题出现了：

> **能不能让数据自己告诉我们，哪个指标更重要？**

今天要学习的，就是解决这类问题的经典方法——**熵权法**。

# 🧠 什么是熵权法？

一句话来说：

> **熵权法，就是利用数据的“差异程度”来确定指标权重的方法。**

它的核心思想非常直观：

> 如果一个指标在所有城市之间差别很大，那么它更重要；  
> 如果一个指标大家都差不多，那么它就不重要。

例如：

- 如果所有城市的“交通指数”几乎一样，那么这个指标对区分城市没有帮助；
    
- 如果“经济水平”差异很大，那么它对评价就非常关键。
    

熵权法本质上是在回答：

> **哪个指标更“有信息量”？**

信息量越大，权重越高。

# 🚩 什么时候想到熵权法？

如果题目中出现下面这些关键词，就可以考虑熵权法：

- 没有专家打分
    
- 权重未知
    
- 客观评价
    
- 数据驱动
    
- 指标差异分析
    
- 信息量
    
- 数据本身决定权重
    

这些问题有一个共同特点：

> **不希望人为决定权重，而希望权重由数据决定。**

例如：

- 城市综合发展评价；
    
- 学校综合排名；
    
- 企业绩效分析；
    
- 产品综合评分。
    

一个简单判断方法是：

> **是否希望“用数据自动确定权重”？**

如果答案是肯定的，那么熵权法就是一个非常自然的选择。

# 🏆 一个完整案例

我们仍然以城市评价为例。

现在有三个城市，从四个指标进行评价：

|城市|GDP|空气质量|交通指数|教育水平|
|---|--:|--:|--:|--:|
|A|12000|80|70|85|
|B|15000|60|75|80|
|C|13000|78|72|82|

今天需要决定什么？

答案很简单：

> **哪个城市综合发展最好。**

但问题在于：

每个指标的重要性是不确定的。

如果让专家打分，这是AHP；但如果没有专家，我们就需要熵权法。

---

## 核心思想：看“差异”

先看一个直觉：

- GDP差距大 → 很重要
    
- 教育差距小 → 影响小
    

熵权法就是量化这个“差异”。

---

# 🧠 熵权法建模过程（思维版）

我们不直接上公式，而是按思路走：

---

## 第一步：标准化

因为指标单位不同：

- GDP（万元）
    
- 空气质量（指数）
    
- 教育水平（分）
    

必须统一尺度。

---

## 第二步：计算占比

把每个城市在某个指标中的比例算出来。

例如：

$$  
p_{ij}  
$$

表示第 $i$ 个城市在第 $j$ 个指标上的占比。

---

## 第三步：计算“信息熵”

如果某个指标：

- 每个城市都差不多 → 信息少
    
- 差异很大 → 信息多
    

熵的公式：

$$  
E_j = -k \sum p_{ij}\ln p_{ij}  
$$

---

## 第四步：计算权重

差异越大，权重越高：

$$  
w_j = \frac{1 - E_j}{\sum (1 - E_j)}  
$$

---

# 💻 Python 实现

```python
import numpy as np

# 原始数据
X = np.array([
    [12000, 80, 70, 85],
    [15000, 60, 75, 80],
    [13000, 78, 72, 82]
], dtype=float)

# 1. 标准化（极差标准化）
X_norm = (X - X.min(axis=0)) / (X.max(axis=0) - X.min(axis=0))

# 2. 防止log(0)
X_norm = X_norm + 1e-12

# 3. 计算比例
P = X_norm / X_norm.sum(axis=0)

# 4. 熵值
n = X.shape[0]
k = 1 / np.log(n)
E = -k * (P * np.log(P)).sum(axis=0)

# 5. 差异系数
d = 1 - E

# 6. 权重
w = d / d.sum()

print("各指标权重：")
for i, wi in enumerate(w):
    print(f"指标{i+1}: {wi:.4f}")
```

# 📌 本章小结

熵权法解决的是一个核心问题：

> **当没有经验时，如何让数据自己决定权重？**

学习熵权法时，可以记住三个关键词：

1. **标准化（统一尺度）**
    
2. **差异（信息量）**
    
3. **权重（由差异决定）**
    

如果一个问题：

- 没有专家意见
    
- 需要客观评价
    
- 权重不确定
    

那么熵权法就是非常自然的选择。

---

> **AHP用经验判断重要性，熵权法用数据判断重要性。**
