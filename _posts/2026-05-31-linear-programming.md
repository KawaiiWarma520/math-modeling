## 🧠 基本概念

**一句话定义**：在资源有限且所有关系成"正比"的条件下，找到使目标（利润/成本/时间等）达到最优的方案。

**三个核心要素**：
- **决策变量**：你要决定的事情（如生产多少个产品）
- **目标函数**：你要最大化或最小化的指标（如总利润）
- **约束条件**：限制你的"天花板"（如不能超过总工时）

**"线性"的含义**：所有变量都是一次方，没有 $x^2$、没有 $\log x$、没有 $x_1 \times x_2$，只有"系数 × 变量"的相加。

## 🏆 经典例子

**生产计划问题**：

> 工厂生产桌子和椅子。桌子利润50元/张，需2小时木工+1小时油漆；椅子利润30元/把，需1小时木工+1小时油漆。每天木工最多8小时，油漆最多5小时。如何安排利润最大？

**模型表达**：
- 决策变量：$x_1$ = 桌子数量，$x_2$ = 椅子数量
- 目标函数：$\max Z = 50x_1 + 30x_2$
- 约束条件：
  - 木工工时：$2x_1 + x_2 \leq 8$
  - 油漆工时：$x_1 + x_2 \leq 5$
  - 非负约束：$x_1 \geq 0, x_2 \geq 0$

## 📊 适用场景

满足以下三个条件即可考虑线性规划：

| 条件 | 通俗解释 | 国赛/美赛例子 |
| :--- | :--- | :--- |
| 有明确目标 | 想最大化或最小化某个东西 | 最大利润、最小成本、最短时间 |
| 资源有限 | 某些东西不能无限使用 | 资金、人力、时间、原材料、运力 |
| 关系线性 | 做2倍的事，消耗2倍的资源 | 不能有"规模折扣" |

**典型问题类型**：
- 生产计划：产品组合优化
- 运输调配：最低运费方案
- 人员排班：最小用工成本
- 投资组合：风险约束下收益最大
- 配料问题：最低成本的营养配餐

## 🛠️ 解题步骤

**五步建模法**：

**Step 1：确定决策变量**
> 问自己：我要决定什么？
> 写法：设 $x_1, x_2, ..., x_n$ 分别代表...

**Step 2：确定目标函数**
> 问自己：我要最大化还是最小化？公式是什么？
> 写法：$\max (\text{或} \min) Z = c_1x_1 + c_2x_2 + ... + c_nx_n$

**Step 3：找出所有约束条件**
> 问自己：什么资源是有限的？有什么硬性要求？
> 写法：$a_{11}x_1 + a_{12}x_2 + ... \leq (\text{或} \geq, =) b_1$

**Step 4：加上非负约束**
> 写法：$x_1, x_2, ..., x_n \geq 0$

**Step 5：求解并解释结果**
> 用工具求解后，回答：最优解是什么？目标值是多少？哪个资源最紧缺？

## ⚠️ 注意事项

**常见误区**：

1. **非线性关系伪装成线性**
   - 错误：把 $x_1 \times x_2$ 当成线性
   - 正确：线性规划不允许变量相乘

2. **忘记非负约束**
   - 错误：只写资源约束，忘了 $x \geq 0$
   - 正确：所有决策变量都要有非负约束

3. **约束方向写反**
   - 错误：资源有限写成 $\geq$（要求比资源还多）
   - 正确：资源约束用 $\leq$，需求约束用 $\geq$

4. **忽略整数要求**
   - 问题：如果决策变量必须是整数（如人数、机器台数），线性规划可能给出小数解
   - 解决：改用**整数规划**

**线性规划的局限性**：
- 只能处理线性关系（现实很多是非线性的）
- 对数据波动敏感（最优解可能随参数微小变化而剧变）
- 无法处理多目标冲突（需改用多目标规划）

## ✏️ 论文写法

**摘要中的表述**：
> 本文建立了基于线性规划的生产优化模型。以最大化总利润为目标函数，以原材料供应量、生产工时为主要约束条件，求解得到最优生产方案为...

**模型建立部分的模板**：

> **决策变量**：设 $x_{ij}$ 表示从第 $i$ 个供应点运往第 $j$ 个需求点的货物数量。
>
> **目标函数**：以总运输成本最小化为目标，建立如下目标函数：
> $$\min Z = \sum_{i=1}^{m}\sum_{j=1}^{n} c_{ij}x_{ij}$$
> 其中 $c_{ij}$ 为单位运输成本。
>
> **约束条件**：
> 1. 供应量约束：$\sum_{j=1}^{n} x_{ij} \leq S_i, \quad i=1,2,...,m$
> 2. 需求量约束：$\sum_{i=1}^{m} x_{ij} \geq D_j, \quad j=1,2,...,n$
> 3. 非负约束：$x_{ij} \geq 0$

**结果分析中的亮点（加分项）**：

> **灵敏度分析**：由对偶变量（影子价格）可知，木工工时每增加1小时，总利润可增加15元，而油漆工时仅增加5元，表明木工是制约生产的瓶颈资源。

> **模型评价**：线性规划模型计算效率高、结果可解释性强，适用于大规模资源分配问题。但其线性假设在实际生产中可能存在偏差，后续可引入非线性规划进行对比验证。

## 💻 代码实现

```python
import pulp

# 1. 创建问题实例（"Max"表示最大化，"MyProblem"是问题名称）
model = pulp.LpProblem("MyProblem", pulp.LpMaximize)

# 2. 定义决策变量（lowBound=0 表示非负，cat='Continuous' 表示连续变量）
x1 = pulp.LpVariable('x1', lowBound=0, cat='Continuous')  # 桌子数量
x2 = pulp.LpVariable('x2', lowBound=0, cat='Continuous')  # 椅子数量

# 3. 定义目标函数
model += 50 * x1 + 30 * x2, "Total_Profit"

# 4. 添加约束条件
model += 2 * x1 + 1 * x2 <= 8, "Woodworking_Time"   # 木工约束
model += 1 * x1 + 1 * x2 <= 5, "Painting_Time"      # 油漆约束

# 5. 求解
model.solve()

# 6. 输出结果
print("求解状态:", pulp.LpStatus[model.status])
print("最优解：")
print(f"  桌子数量 x1 = {x1.varValue}")
print(f"  椅子数量 x2 = {x2.varValue}")
print(f"  最大利润 = {pulp.value(model.objective)} 元")
```

## 🔗 关联模型

| 模型 | 关系 | 何时升级使用 |
| --------- | ----------------- | ------------------- |
| **整数规划** | 线性规划的"整数版" | 当变量必须是整数时（如人数、机器数） |
| **0-1规划** | 整数规划的特例（变量只能取0或1） | 做"是/否"决策（如是否建仓库） |
| **多目标规划** | 线性规划的"多目标版" | 需要同时优化多个冲突目标时 |
| **运输问题** | 线性规划的特殊形式 | 物流、配送类问题 |
| **目标规划** | 线性规划的"软约束版" | 约束可以"尽量满足"而不是"必须满足" |
| **非线性规划** | 线性规划的"非线性版" | 存在规模经济、边际效益递减等情况 |
