## 🧠 基本概念

**一句话定义**：多目标规划 = 在多个相互冲突的目标之间，找到一个"平衡"的满意解。

**与线性规划的区别**：

| 维度 | 线性规划 | 多目标规划 |
| :--- | :--- | :--- |
| 目标数量 | 单个 | 两个及以上 |
| 解的类型 | 最优解（唯一或有限几个） | 帕累托最优解（一组解） |
| 核心问题 | 找到"最好"的方案 | 找到"最平衡"的方案 |

**核心概念：帕累托最优**

> 当一个方案无法在"不损害其他目标"的前提下改进任何一个目标时，该方案就是帕累托最优的。

通俗理解：想再提高利润，就必须接受更多污染；想再减少污染，就必须接受更低利润。这就是"权衡"（trade-off）。

**三种常见类型**：

| 类型 | 说明 | 适用场景 |
|:---|:---|:---|
| 加权法 | 给每个目标赋权重，相加成单目标 | 知道各目标的重要程度 |
| 约束法（epsilon约束） | 保留一个主目标，其余变成约束 | 有明确"底线要求" |
| 理想点法 | 先算各目标理想值，再找离理想值最近的点 | 想平衡所有目标 |

## 🏆 经典例子

**工厂生产问题**：

> 工厂生产两种产品，面临三个冲突目标：
> - 目标1：利润最大化（元）
> - 目标2：污染最小化（吨）
> - 目标3：员工加班时间最小化（小时）

| 方案 | 利润（元） | 污染（吨） | 加班（小时） |
|:---|:---|:---|:---|
| 方案A | 8000 | 4 | 10 |
| 方案B | 6000 | 2 | 6 |
| 方案C | 7000 | 3 | 8 |

三个方案各有优劣：没有哪个方案在所有目标上都优于其他方案。这就是多目标规划要处理的典型场景。

## 📊 适用场景

| 场景 | 冲突的目标 |
| :--- | :--- |
| 生产计划 | 利润最大 vs 成本最小 vs 污染最小 |
| 物流配送 | 成本最小 vs 时间最短 vs 车辆最少 |
| 投资组合 | 收益最大 vs 风险最小 |
| 选址问题 | 覆盖人口最多 vs 建设成本最小 |
| 水资源分配 | 农业用水 vs 工业用水 vs 生态用水 |
| 能源调度 | 发电成本最小 vs 碳排放最小 |

**比赛中常见的多目标表述**：
- "在保证...的前提下，尽可能..."
- "综合考虑...和...，提出一个平衡方案"
- "既要...又要...，请给出最优决策"

## 🛠️ 解题步骤

以**加权法**为例（最常用、最简单）：

**Step 1：定义决策变量**
- 和线性规划一样，设 $x_1, x_2, ...$ 表示要决定的事情

**Step 2：写出所有目标函数**
- 目标1：$f_1 = ...$（如利润最大）
- 目标2：$f_2 = ...$（如污染最小）
- 目标3：$f_3 = ...$（如加班最小）

**Step 3：确定约束条件**
- 和线性规划一样，写出资源限制等

**Step 4：归一化 + 加权（关键步骤）**

*4.1 归一化（消除量纲）*

根据目标方向选择公式：

> **越大越好（利润）**
> 公式：$f' = \frac{f - f_{min}}{f_{max} - f_{min}}$
> 结果：越好 → 越接近 1

> **越小越好（污染）**
> 公式：$f' = \frac{f_{max} - f}{f_{max} - f_{min}}$
> 结果：越好 → 越接近 1

*4.2 加权求和*

$$\min (\text{或} \max) Z = w_1 f_1' + w_2 f_2' + w_3 f_3'$$

其中 $w_1 + w_2 + w_3 = 1$，$w_i \geq 0$

**Step 5：求解并解释结果**
- 求解后得到一组决策变量值
- 解释：这个方案在各目标上的表现如何？为什么是"平衡"的？

## ⚠️ 注意事项

**1. 必须归一化**
- 不同量纲的数据不能直接相加（如 5000 元 + 4 吨）
- 不归一化会导致数值大的目标"吃掉"数值小的目标

**2. 注意目标方向**
- 加权前必须统一方向（都用"越大越好"或都用"越小越好"）
- 归一化时已处理，公式已区分

**3. 权重的设定**
- 可以主观设定（根据问题重要性）
- 可以用层次分析法 AHP 计算权重
- 可以用熵权法客观计算
- 建议做灵敏度分析：改变权重，观察结果变化

**4. 加权法的局限性**
- 权重主观性较强
- 无法得到完整的帕累托前沿（只能得到一个点）

**5. 约束法的使用场景**
- 当有明确的"底线"要求时（如污染不能超过 5 吨）
- 可以把次要目标变成约束，只优化最重要的目标

## ✏️ 论文写法

**摘要中的表述**：
> 本文建立了多目标规划模型。以利润最大化、污染最小化、加班时间最小化为目标，采用加权法将多目标转化为单目标，通过层次分析法确定权重，求解得到满意解为...

**模型建立部分的模板**：

> **决策变量**：设 $x_1, x_2$ 分别表示产品 A 和产品 B 的生产数量。
>
> **目标函数**：本文考虑三个目标：
> 1. 利润最大化：$\max f_1 = 50x_1 + 30x_2$
> 2. 污染最小化：$\min f_2 = 2x_1 + x_2$
> 3. 加班最小化：$\min f_3 = x_1 + 2x_2$
>
> **归一化处理**：由于各目标量纲不同，采用极差归一化：
> - 利润（越大越好）：$f_1' = \frac{f_1 - f_{1min}}{f_{1max} - f_{1min}}$
> - 污染（越小越好）：$f_2' = \frac{f_{2max} - f_2}{f_{2max} - f_{2min}}$
> - 加班（越小越好）：$f_3' = \frac{f_{3max} - f_3}{f_{3max} - f_{3min}}$
>
> **加权求和**：设权重向量 $w = (0.5, 0.3, 0.2)$，转化为单目标：
> $$\max Z = 0.5 f_1' + 0.3 f_2' + 0.2 f_3'$$

**结果分析中的亮点**：

> 通过灵敏度分析，我们改变污染目标的权重，发现当 $w_2 \leq 0.2$ 时，最优解偏向高利润方案；当 $w_2 \geq 0.4$ 时，最优解偏向低污染方案。决策者可根据实际偏好选择权重。

> 采用约束法，将污染上限分别设为 3 吨、4 吨、5 吨，得到利润的帕累托前沿曲线，展示了"利润-污染"之间的权衡关系。

## 💻 代码实现

```python
import pulp

def solve_with_pollution_limit(pollution_limit):
    model = pulp.LpProblem("Constrained", pulp.LpMaximize)
    
    x1 = pulp.LpVariable('x1', lowBound=0)
    x2 = pulp.LpVariable('x2', lowBound=0)
    
    # 只优化利润
    model += 50 * x1 + 30 * x2, "Profit"
    
    # 约束条件
    model += 2 * x1 + x2 <= 8
    model += x1 + x2 <= 5
    model += 2 * x1 + x2 <= pollution_limit  # 污染约束
    
    model.solve()
    return pulp.value(model.objective)

# 画出帕累托前沿
for limit in [3, 4, 5, 6, 7, 8]:
    profit_val = solve_with_pollution_limit(limit)
    print(f"污染上限={limit}吨, 最大利润={profit_val:.2f}元")
```

## 🔗 关联模型

| 模型 | 关系 | 何时使用 |
| :--- | :--- | :--- |
| **线性规划** | 多目标规划的特例（单目标） | 只有一个优化目标时 |
| **目标规划** | 多目标规划的变种 | 目标有优先级，可以"尽量满足" |
| **层次分析法 AHP** | 辅助工具 | 确定各目标的权重 |
| **熵权法** | 辅助工具 | 客观确定权重 |
| **帕累托优化** | 理论基础 | 需要找出一组帕累托最优解时 |
