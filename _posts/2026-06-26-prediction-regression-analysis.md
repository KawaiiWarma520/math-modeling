# 📖 故事引入

一家奶茶品牌在多个城市开店后，发现一个很现实的问题：

有些门店销量很好，有些门店却一直不温不火。

他们收集了数据：

- 广告投入
    
- 门店面积
    
- 人流量
    
- 销量
    

老板想知道一个关键问题：

> 到底什么因素最影响销量？

如果只是看趋势（时间序列），只能预测未来；  
但现在他们更想知道：

> **销量变化到底是由什么“原因”决定的？**

今天要学习的，就是解决这类问题的经典模型——**回归分析**。

---

# 🧠 什么是回归分析？

一句话来说：

> **回归分析，就是用数学函数描述“自变量”与“因变量”之间关系的方法。**

在回归中：

- 因变量：我们要预测的结果（如销量）
    
- 自变量：影响结果的因素（如广告费）
    

例如：

$$  
销量 = f(广告费)  
$$

回归分析的目标是找到这个函数。

---

# 🚩 什么时候想到回归分析？

如果题目中出现下面这些情况，就可以考虑回归分析：

- 想研究影响因素
    
- 想建立变量关系
    
- 想预测结果
    
- 有多个解释变量
    
- 数据是“因果型结构”
    

例如：

- 广告费 → 销量
    
- 温度 → 用电量
    
- 房价 → 面积、地段
    
- 收入 → 教育水平
    

一个简单判断方法：

> **如果题目在问“什么影响了什么”，就应该想到回归分析。**

---

# 🧠 回归分析的核心思想

回归分析本质上做一件事：

> 用一条“最合适的曲线”去拟合数据。

最常见的是线性回归：

$$  
y = ax + b  
$$

如果有多个变量：

$$  
y = a_1x_1 + a_2x_2 + \cdots + a_nx_n + b  
$$

其中：

- 系数 $a_i$ 表示影响程度
    
- $b$ 表示基础水平
    

---

# 🏆 一个完整案例

某奶茶品牌统计了5家门店的数据：

|门店|广告费(千元)|人流量(千人)|销量(千杯)|
|---|--:|--:|--:|
|A|2|10|30|
|B|3|15|40|
|C|4|20|50|
|D|5|22|55|
|E|6|25|65|

---

## 今天要解决的问题：

> 广告费和人流量，到底如何影响销量？

---

## 建立模型

设：

- $x_1$：广告费
    
- $x_2$：人流量
    
- $y$：销量
    

模型：

$$  
y = a_1x_1 + a_2x_2 + b  
$$

---

# 🧠 如何理解这个模型？

它其实在做一件事：

> 把“复杂现实”拆成“线性组合”

例如：

- 广告费每增加1单位 → 销量增加多少？
    
- 人流量每增加1单位 → 销量增加多少？
    

---

# 💻 Python 实现

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 自变量：广告费 + 人流量
X = np.array([
    [2, 10],
    [3, 15],
    [4, 20],
    [5, 22],
    [6, 25]
])

# 因变量：销量
y = np.array([30, 40, 50, 55, 65])

# 建立模型
model = LinearRegression()
model.fit(X, y)

# 输出参数
print("系数：", model.coef_)
print("截距：", model.intercept_)

# 预测
print("预测销量：", model.predict([[7, 28]]))
```

# 📌 本章小结

回归分析解决的是：

> **变量之间“关系未知，但可以通过数据学习”的问题**

学习回归分析时，可以记住三个关键词：

1. **关系（变量之间的联系）**
    
2. **拟合（寻找最优函数）**
    
3. **预测（用函数推未来）**
    

---

## 🔥 和时间序列的区别

|模型|核心问题|
|---|---|
|时间序列|未来趋势|
|回归分析|因果关系|

---

> **时间序列看“时间怎么变”，回归看“什么导致变化”。**
