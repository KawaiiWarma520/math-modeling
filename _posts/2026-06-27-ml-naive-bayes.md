# 📖 故事引入

某邮箱系统每天都要处理大量邮件：

- 正常邮件
    
- 垃圾邮件
    
- 广告邮件
    
- 钓鱼邮件
    

系统需要自动判断：

> 这封邮件是不是垃圾邮件？

但问题是：

- 关键词很多
    
- 特征很多
    
- 组合方式复杂
    

于是工程师提出一个问题：

> **能不能用“概率”来判断一封邮件属于哪一类？**

今天要学习的，就是解决这类问题的经典模型——**朴素贝叶斯（Naive Bayes）**。

---

# 🧠 什么是朴素贝叶斯？

一句话来说：

> **朴素贝叶斯，就是基于贝叶斯公式，并假设各特征之间相互独立的分类模型。**

它本质是：

> 用概率做分类决策

---

# 🚩 什么时候想到朴素贝叶斯？

如果题目中出现下面这些情况，就可以考虑：

- 分类问题
    
- 文本分类
    
- 高维特征
    
- 特征相互影响不明确
    
- 需要快速建模
    

例如：

- 垃圾邮件识别
    
- 情感分析（正面/负面）
    
- 医疗诊断
    
- 风险分类
    

一个简单判断方法：

> **只要是“分类 + 概率 + 多特征”，就可以考虑朴素贝叶斯。**

---

# 🧠 核心思想：贝叶斯公式

基础公式：

$$  
P(C|X) = \frac{P(X|C)P(C)}{P(X)}  
$$

其中：

- $C$：类别
    
- $X$：特征
    

---

# 🧠 “朴素”的含义

关键假设：

> **所有特征相互独立**

即：

$$  
P(X|C) = \prod P(x_i|C)  
$$

---

## 📌 这个假设虽然不真实，但很好用

现实中：

- 特征往往相关
    
- 但模型仍然有效
    

---

# 🧠 朴素贝叶斯在做什么？

一句话理解：

> 比较“哪个类别的概率最大”

---

# 🏆 一个完整案例

某系统要判断邮件是否为垃圾邮件：

我们观察关键词：

- “中奖”
    
- “免费”
    
- “点击”
    

---

## 今天要解决的问题：

> 这封邮件是不是垃圾邮件？

---

## 建模思路

我们计算：

- 是垃圾邮件的概率
    
- 不是垃圾邮件的概率
    

然后选择最大值

---

# 🧠 决策逻辑

$$  
\text{预测类别} = \arg\max P(C|X)  
$$

---

# 💻 Python实现

```python
from sklearn.naive_bayes import GaussianNB
import numpy as np

# 特征：是否包含关键词（简化）
X = np.array([
    [1, 1, 1],  # 垃圾邮件
    [1, 0, 1],  # 垃圾邮件
    [0, 0, 0],  # 正常邮件
    [0, 1, 0],  # 正常邮件
])

# 标签：1=垃圾邮件，0=正常邮件
y = np.array([1, 1, 0, 0])

# 模型
model = GaussianNB()
model.fit(X, y)

# 预测
print(model.predict([[1, 1, 0]]))
```

---

# 🧠 朴素贝叶斯的本质

一句话总结：

> **用概率推理代替复杂建模，用“独立假设”简化计算**

---

# 📌 本章小结

朴素贝叶斯解决的是：

> **基于概率的分类问题**

学习它时，可以记住三个关键词：

1. **贝叶斯公式**
    
2. **特征独立假设**
    
3. **概率最大化分类**
    

---

## 🔥 和 Logistic 的区别

|模型|思路|
|---|---|
|Logistic|判别模型|
|朴素贝叶斯|生成模型|

---

> **Logistic是在“直接分类”，朴素贝叶斯是在“算概率再分类”。**
