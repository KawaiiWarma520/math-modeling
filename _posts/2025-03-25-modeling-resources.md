从入门到进阶，这些资源是我亲身用过且觉得值得推荐的。

## 📖 必读书籍

### 入门篇

| 书名 | 作者 | 推荐理由 |
|------|------|----------|
| 《数学建模算法与应用》 | 司守奎 | 国内最经典的建模教材，案例丰富，代码完整 |
| 《数学模型》 | 姜启源 | 理论扎实，适合打基础 |
| 《数学建模方法与分析》 | Mark M. Meerschaert | 国外经典教材，思路清晰 |

### 进阶篇

- **《统计学习方法》** — 李航：机器学习入门必备
- **《数值分析》** — 适用于需要高精度计算的场景
- **《MATLAB 在数学建模中的应用》** — 代码实战导向

## 🎬 优质网课

1. **中国大学 MOOC** — 数学建模（西北工业大学）
2. **b站 UP 主** — 搜索"数学建模"有很多宝藏up
3. **Coursera** — *Mathematics for Machine Learning* 系列

## 📁 必备资源清单

### 比赛期间常备

- [x] 论文模板（Word + LaTeX 各一份）
- [x] 常用算法代码库（MATLAB + Python）
- [x] 数据预处理脚本
- [x] 可视化模板

### 数据源推荐

- **Kaggle** — 丰富的数据集和竞赛
- **国家统计局** — 官方数据权威可靠
- **阿里天池** — 国内数据竞赛平台
- **Data.gov** — 美国政府开放数据

## ⭐ 实用工具链

```
问题分析 → 模型建立 → 求解计算 → 结果分析 → 论文撰写
   ↓           ↓           ↓           ↓           ↓
  XMind      LaTeX      MATLAB       Python      Word/
  (思维导图)  (公式)     Python      (可视化)    LaTeX
```

## 💡 我的建议

1. **不要贪多**：把 5-8 种经典模型吃透，远胜于泛泛了解 20 种
2. **代码库要慢慢积累**：每做一道题，就把代码整理好
3. **多看优秀论文**：O奖论文是学习写作的最佳范本
