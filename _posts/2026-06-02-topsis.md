## 🧠 基本概念

**一句话定义**：TOPSIS（逼近理想解排序法）是一种通过计算各方案与“正理想解”和“负理想解”的距离，得到相对贴近度，从而进行优劣排序的多指标决策方法。

**核心思想**：
- **正理想解**：各指标都达到最好（虚拟的最优方案）
- **负理想解**：各指标都达到最差（虚拟的最劣方案）
- 最优方案 = 离正理想解最近 + 离负理想解最远

**相对贴近度公式**：
$$C_i = \frac{D_i^-}{D_i^+ + D_i^-}$$

- $C_i$ 越接近 1 → 方案越优
- $C_i$ 越接近 0 → 方案越差

- 任何“给一堆对象打分排序”的问题

|:---|:---|:---|:---|

## 🛠️ 解题步骤

### Step 1：构造决策矩阵

$n$ 个方案，$m$ 个指标：

$$
X = \begin{bmatrix}
x_{11} & x_{12} & \cdots & x_{1m} \\
x_{21} & x_{22} & \cdots & x_{2m} \\
\vdots & \vdots & \ddots & \vdots \\
x_{n1} & x_{n2} & \cdots & x_{nm}
\end{bmatrix}
$$

### Step 2：向量归一化（消除量纲）

$$r_{ij} = \frac{x_{ij}}{\sqrt{\sum_{i=1}^{n} x_{ij}^2}}$$

得到归一化矩阵 $R$。

**为什么用向量归一化？**
- 保留数据间的相对比例
- 适合后续计算欧氏距离

### Step 3：构造加权归一化矩阵

$$v_{ij} = w_j \times r_{ij}$$

其中 $w_j$ 是第 $j$ 个指标的权重（可用熵权法或 AHP 得到）。

### Step 4：确定正理想解 $A^+$ 和负理想解 $A^-$

$$A^+ = \{ v_1^+, v_2^+, ..., v_m^+ \}$$
$$A^- = \{ v_1^-, v_2^-, ..., v_m^- \}$$

对于第 $j$ 个指标：

| 指标类型 | $v_j^+$（正理想解） | $v_j^-$（负理想解） |
|:---|:---|:---|
| 越大越好 | $\max(v_{1j},...,v_{nj})$ | $\min(v_{1j},...,v_{nj})$ |
| 越小越好 | $\min(v_{1j},...,v_{nj})$ | $\max(v_{1j},...,v_{nj})$ |

### Step 5：计算各方案到正、负理想解的距离（欧氏距离）

$$D_i^+ = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^+)^2}$$
$$D_i^- = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^-)^2}$$

### Step 6：计算相对贴近度

$$C_i = \frac{D_i^-}{D_i^+ + D_i^-}$$

### Step 7：按 $C_i$ 从大到小排序

$C_i$ 越大，方案越优。

---

**经典例子完整计算**：

| 手机 | 价格 | 性能 | 续航 |
|:---|:---|:---|:---|
| iPhone | 8000 | 95 | 10 |
| 华为 | 5000 | 90 | 12 |
| 小米 | 3000 | 80 | 8 |

假设权重相等 $w = (1/3, 1/3, 1/3)$，计算后得到：

| 手机 | $D^+$ | $D^-$ | $C$ | 排名 |
|:---|:---|:---|:---|:---|
| 小米 | 0.12 | 0.35 | 0.74 | 1 |
| 华为 | 0.28 | 0.22 | 0.44 | 2 |
| iPhone | 0.38 | 0.10 | 0.21 | 3 |

→ 小米得分最高，推荐小米。

## ⚠️ 注意事项

**1. 权重来源要交代清楚**
- 用熵权法：说明“基于数据客观计算”
- 用 AHP：说明“基于专家打分”
- 两者组合：说明“兼顾主客观”

**2. 归一化方法要写明**
- 推荐使用向量归一化：$r_{ij} = x_{ij} / \sqrt{\sum x_{ij}^2}$
- 也可用极差归一化，但需在论文中说明

**3. TOPSIS 的局限性**
- 需要各指标相互独立（不独立时考虑其他方法）
- 权重对结果影响较大
- 正负理想解依赖数据范围（新增方案时需要重新计算）

**4. 与熵权法的经典组合**

| 步骤 | 方法 | 产出 |
|:---|:---|:---|
| 1 | 熵权法（或 AHP） | 各指标权重 $w_j$ |
| 2 | TOPSIS | 各方案的相对贴近度 $C_i$ |
| 3 | 排序 | 选出最优方案 |

**5. 可视化建议**
- 画雷达图展示各方案多维度对比
- 画柱状图展示各方案的 $D^+$、$D^-$、$C$ 值
- 画散点图（以 $D^+$ 为横轴、$D^-$ 为纵轴）展示方案分布

**模型建立部分的模板**：

> **1. 构造决策矩阵**
>
> 设有 $n$ 个评价对象，$m$ 个评价指标，构造决策矩阵 $X = (x_{ij})_{n \times m}$。
>
> **2. 向量归一化**
>
> $$r_{ij} = \frac{x_{ij}}{\sqrt{\sum_{i=1}^{n} x_{ij}^2}}, \quad i=1,...,n; j=1,...,m$$
>
> **3. 构造加权归一化矩阵**
>
> $$v_{ij} = w_j \times r_{ij}$$
>
> 其中 $w_j$ 为第 $j$ 个指标的权重，由熵权法（AHP）确定。
>
> **4. 确定正负理想解**
>
> 正理想解 $A^+ = (v_1^+, v_2^+, ..., v_m^+)$，负理想解 $A^- = (v_1^-, v_2^-, ..., v_m^-)$，其中：
> - 对于越大越好的指标：$v_j^+ = \max_i v_{ij}$，$v_j^- = \min_i v_{ij}$
> - 对于越小越好的指标：$v_j^+ = \min_i v_{ij}$，$v_j^- = \max_i v_{ij}$
>
> **5. 计算距离**
>
> $$D_i^+ = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^+)^2}, \quad D_i^- = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^-)^2}$$
>
> **6. 计算相对贴近度**
>
> $$C_i = \frac{D_i^-}{D_i^+ + D_i^-}$$
>
> **7. 排序**
>
> 按 $C_i$ 从大到小排序，$C_i$ 越大表示方案越优。

**结果分析中的表述**：
> 由 TOPSIS 计算结果可知，方案 A 的相对贴近度为 0.82，排名第一；方案 B 为 0.45，排名第二；方案 C 为 0.23，排名第三。因此推荐方案 A。

**与熵权法组合的表述**：
> 本文采用熵权法确定指标权重（客观赋权），进而运用 TOPSIS 方法计算各评价对象的相对贴近度。这种组合既避免了主观偏差，又实现了多指标的综合排序。

## 💻 代码实现


```Python
import numpy as np
import pandas as pd

def topsis(data, weights, directions):
    """
    TOPSIS 方法计算相对贴近度
    data: numpy array, 形状 (n_samples, n_features)
    weights: 权重列表，长度 m
    directions: 指标方向列表，'positive' 越大越好，'negative' 越小越好
    """
    X = np.array(data)
    n, m = X.shape
    
    # Step 2: 向量归一化
    norm = np.sqrt(np.sum(X**2, axis=0))
    norm[norm == 0] = 1  # 避免除零
    R = X / norm
    
    # Step 3: 加权归一化矩阵
    V = R * np.array(weights)
    
    # Step 4: 确定正负理想解
    ideal_best = []
    ideal_worst = []
    for j in range(m):
        if directions[j] == 'positive':
            ideal_best.append(np.max(V[:, j]))
            ideal_worst.append(np.min(V[:, j]))
        else:  # negative
            ideal_best.append(np.min(V[:, j]))
            ideal_worst.append(np.max(V[:, j]))
    
    # Step 5: 计算距离
    D_plus = np.sqrt(np.sum((V - ideal_best)**2, axis=1))
    D_minus = np.sqrt(np.sum((V - ideal_worst)**2, axis=1))
    
    # Step 6: 相对贴近度
    C = D_minus / (D_plus + D_minus)
    
    return C, D_plus, D_minus

# ========== 例子：手机选购 ==========
data = np.array([
    [8000, 95, 10],  # iPhone
    [5000, 90, 12],  # 华为
    [3000, 80, 8]    # 小米
])

weights = [1/3, 1/3, 1/3]  # 等权重
directions = ['negative', 'positive', 'positive']  # 价格越小越好，性能/续航越大越好

C, D_plus, D_minus = topsis(data, weights, directions)

print("=== TOPSIS 计算结果 ===")
print(f"正理想解距离 D+: {D_plus}")
print(f"负理想解距离 D-: {D_minus}")
print(f"相对贴近度 C: {C}")
print(f"排名: {np.argsort(-C) + 1}")  # 从大到小排序

# 结果展示
results = pd.DataFrame({
    '方案': ['iPhone', '华为', '小米'],
    'D+': D_plus,
    'D-': D_minus,
    'C值': C
})
results = results.sort_values('C值', ascending=False)
print("\n最终排名：")
print(results.to_string(index=False))
```

## 🔗 关联模型

|模型|关系|何时使用|
|---|---|---|
|**熵权法**|客观赋权法|与 TOPSIS 组成经典组合，先赋权后排|
|**AHP**|主观赋权法|需要主观判断时，为 TOPSIS 提供权重|
|**多目标规划**|优化方法|需要找到具体决策变量值时使用|
|**主成分分析 (PCA)**|降维方法|指标过多时可先降维，再做 TOPSIS|
|**灰色关联分析**|评价方法|另一种综合评价方法，可与 TOPSIS 对比|
