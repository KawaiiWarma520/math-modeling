# 📖 故事引入

某研究机构正在评估几个城市的发展质量。

他们已经收集了多个指标：

- GDP
    
- 人均收入
    
- 空气质量
    
- 交通指数
    
- 教育水平
    

一开始，他们使用了熵权法。

结果发现一个奇怪现象：

有些看起来“非常相关”的指标，被赋予了几乎一样的重要性。

例如：

- GDP 和人均收入几乎同步变化；
    
- 交通指数和空气质量也存在一定关联。
    

于是研究人员提出了一个问题：

> **如果两个指标高度相似，那它们真的应该“都很重要”吗？**

今天要学习的，就是解决这个问题的模型——**CRITIC权重法**。

# 🧠 什么是 CRITIC？

一句话来说：

> **CRITIC方法 = 指标的“波动性” + 指标之间的“冲突性”**

它和熵权法最大的不同是：

|方法|只看什么|
|---|---|
|熵权法|数据差异|
|CRITIC|差异 + 相关性|

CRITIC认为：

> 一个好的指标，不仅要“有变化”，还要“和其他指标不一样”。

---

# 🚩 什么时候想到 CRITIC？

如果题目中出现下面这些情况，就可以考虑 CRITIC：

- 指标之间可能存在相关性
    
- 数据冗余较多
    
- 需要更“科学”的权重
    
- 熵权法结果不够合理
    
- 想提升模型严谨性
    

这些问题的核心特点是：

> **不是每个指标都是独立的**

例如：

- GDP 和收入（高度相关）
    
- 教育水平和科研水平（部分相关）
    
- 交通和经济（弱相关）
    

如果这些相关性不处理，就可能导致：

> 信息重复计算，权重失真

---

# 🧠 CRITIC 的核心思想

CRITIC认为，一个指标的重要性来自两个部分：

---

## ① 波动性（标准差）

如果一个指标变化很小：

→ 区分能力弱  
→ 不重要

例如：

空气质量在所有城市几乎一样 → 没有区分意义

---

## ② 冲突性（相关性）

如果一个指标和其他指标高度一致：

→ 信息重复  
→ 重要性下降

例如：

GDP 和收入几乎同步变化 → 信息冗余

---

## 合在一起：

$$  
C_j = \sigma_j \cdot \sum (1 - r_{jk})  
$$

其中：

- $\sigma_j$：标准差（波动性）
    
- $r_{jk}$：相关系数
    
- $C_j$：综合信息量
    

最终权重：

$$  
w_j = \frac{C_j}{\sum C_j}  
$$

---

# 🏆 一个完整案例

我们仍然使用城市评价问题。

|城市|GDP|收入|空气质量|教育水平|
|---|--:|--:|--:|--:|
|A|12000|35000|80|85|
|B|15000|42000|60|80|
|C|13000|38000|78|82|

---

## 今天要解决的问题：

> 哪个指标更“值得被重视”？

---

## CRITIC的直觉过程

### 第一步：看波动

- GDP变化大 → 有信息
    
- 空气质量变化较小 → 信息弱
    

---

### 第二步：看冲突

- GDP 和收入 → 强相关（信息重复）
    
- 教育 → 相对独立
    

---

### 第三步：综合判断

一个指标如果：

- 波动大 ✔
    
- 且不和其他指标重复 ✔
    

→ 权重就高

---

# 💻 Python 实现

```python
import numpy as np

# 原始数据
X = np.array([
    [12000, 35000, 80, 85],
    [15000, 42000, 60, 80],
    [13000, 38000, 78, 82]
], dtype=float)

# 1. 标准化
X_norm = (X - X.min(axis=0)) / (X.max(axis=0) - X.min(axis=0))

# 2. 标准差（波动性）
std = X_norm.std(axis=0)

# 3. 相关系数矩阵
corr = np.corrcoef(X_norm, rowvar=False)

# 4. 冲突性
conflict = (1 - corr).sum(axis=1)

# 5. 信息量
C = std * conflict

# 6. 权重
w = C / C.sum()

print("CRITIC权重：")
for i, wi in enumerate(w):
    print(f"指标{i+1}: {wi:.4f}")
```

# 📌 本章小结

CRITIC解决的是一个更现实的问题：

> **指标之间不是独立的，它们会“互相影响”。**

学习CRITIC时，可以记住三个关键词：

1. **波动性（标准差）**
    
2. **冲突性（相关性）**
    
3. **信息量（两者结合）**
    

---

## 🔥 和熵权法的核心区别

|方法|核心思想|
|---|---|
|熵权法|谁“变化大”谁重要|
|CRITIC|谁“变化大 + 不重复”谁重要|

---

> **熵权法看“信息多少”，CRITIC看“信息是否重复”。**
