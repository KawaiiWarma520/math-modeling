## 🧠 基本概念

**一句话定义**：多目标规划 = 在多个相互冲突的目标之间，找到一个"平衡"的满意解。

**与线性规划的区别**：

| 维度 | 线性规划 | 多目标规划 |
| :--- | :--- | :--- |
| 目标数量 | 单个 | 两个及以上 |
| 解的类型 | 最优解（唯一或有限几个） | 帕累托最优解（一组解） |
| 核心问题 | 找到"最好"的方案 | 找到"最平衡"的方案 |

**核心概念：帕累托最优**

> 如果一个方案在不损害其他目标的前提下，无法再改善任何一个目标，那它就是**帕累托最优**的。

## 🏆 经典例子

> 工厂生产两种产品，要同时最大化利润和最小化污染。利润越高越好，污染越低越好——这两个目标是冲突的。

**模型表达**：
- 决策变量：$x_1, x_2$ 为两种产品的产量
- 目标1：$\max f_1 = 50x_1 + 30x_2$（利润）
- 目标2：$\min f_2 = 5x_1 + 2x_2$（污染）
- 约束：原材料、工时等限制

## 🛠️ 常用求解方法

### 方法一：加权法

把多个目标加权求和变成一个目标：

$$\max Z = w_1 \cdot f_1 - w_2 \cdot f_2$$

权重 $w_1, w_2$ 由决策者根据偏好设定。

### 方法二：$\varepsilon$ 约束法

把其中一个目标作为主目标，其他目标转化为约束：

> $\max f_1$，且 $f_2 \leq \varepsilon$

通过改变 $\varepsilon$ 的值，可以得到一组帕累托解。

### 方法三：理想点法

先求出每个目标单独最优的值（理想点），然后找距离理想点最近的解：

$$\min D = \sqrt{(f_1 - f_1^*)^2 + (f_2 - f_2^*)^2}$$

## 💻 代码实现（加权法）

```python
import pulp

model = pulp.LpProblem("MultiObjective", pulp.LpMaximize)

x1 = pulp.LpVariable('x1', lowBound=0)
x2 = pulp.LpVariable('x2', lowBound=0)

# 原始目标系数
f1_coeff = [50, 30]  # 利润
f2_coeff = [5, 2]    # 污染（越小越好）

# 权重（决策者偏好）
w1, w2 = 0.7, 0.3

# 归一化后合并为目标函数
model += w1 * (50*x1 + 30*x2) - w2 * (5*x1 + 2*x2)

# 约束
model += 2*x1 + x2 <= 8
model += x1 + x2 <= 5

model.solve()
print(f"x1 = {x1.varValue}, x2 = {x2.varValue}")
print(f"目标值 = {pulp.value(model.objective)}")
```
