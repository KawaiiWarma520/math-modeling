# 📖 一、题目长什么样？

某省要评选“最优发展城市”，给出了以下指标：

- GDP
    
- 环境质量
    
- 教育水平
    
- 医疗资源
    
- 交通便利度
    

问题是：

> ❗ 每个城市都不一样，怎么判断谁更好？

---

# 🧠 二、第一步：这是什么问题？

先别急着选模型，先判断类型：

关键词：

- “评选”
    
- “综合评价”
    
- “多个指标”
    
- “没有唯一答案”
    

---

## 📌 结论：

> ✔ 这是典型 **评价问题**

---

# 🧠 三、第二步：评价问题本质是什么？

一句话：

> **把多个指标 → 转换成一个“可比较的分数”**

---

所以核心问题变成：

> ❗ 怎么“合理加权 + 排序”？

---

# 📊 四、第三步：建模流程拆解

评价模型通常分三步：

---

## 📌 ① 构建指标体系

例如：

|城市|GDP|环境|教育|医疗|
|---|---|---|---|---|

👉 原始数据矩阵

---

## 📌 ② 确定权重

这是评价模型核心🔥

---

### ✔ 方法1：AHP（主观）

- 专家打分
    
- 构造判断矩阵
    

---

### ✔ 方法2：熵权法（客观）

- 数据越分散 → 权重越高
    

---

### ✔ 方法3：CRITIC

- 考虑相关性 + 波动性
    

---

👉 结论：

> 权重 = “重要性分配问题”

---

## 📌 ③ 综合评分 / 排序

---

### ✔ TOPSIS核心思想：

> 距离“最优解”最近的最好

---

### ✔ DEA思想：

> 谁“最有效率”

---

# 💻 七、Python示例（TOPSIS简化版）

```python
import numpy as np

# 原始数据（城市×指标）
X = np.array([
    [100, 80, 90],
    [90, 85, 88],
    [80, 70, 95]
])

# 简单归一化
X_norm = X / np.sqrt((X**2).sum(axis=0))

# 假设权重
w = np.array([0.4, 0.3, 0.3])

# 加权
X_weighted = X_norm * w

# 理想解
best = X_weighted.max(axis=0)
worst = X_weighted.min(axis=0)

# 距离
d_best = np.sqrt(((X_weighted - best)**2).sum(axis=1))
d_worst = np.sqrt(((X_weighted - worst)**2).sum(axis=1))

# 相对接近度
score = d_worst / (d_best + d_worst)

print("评价得分：", score)
```
