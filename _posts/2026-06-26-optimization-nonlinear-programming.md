# 📖 故事引入

一家新开的奶茶店准备在线上投放广告。

老板手里有一笔固定的宣传预算，可以投放在短视频平台，也可以投放在本地生活平台。

刚开始，广告效果非常明显，投入一点预算，就能吸引不少顾客。

但随着预算不断增加，老板发现了一个奇怪的现象：继续投入广告，新增顾客却越来越少。

这就是现实世界中常见的**边际效应递减**。

广告投放越多，并不意味着收益会一直按相同速度增长。

这时候，再使用线性规划就不合适了。

今天要学习的，就是能够处理这类问题的模型——**非线性规划**。

# 🧠 什么是非线性规划？

一句话来说：

> **非线性规划，就是目标函数或约束条件中出现非线性关系的优化模型。**

在线性规划中，我们要求目标函数和约束条件都是变量的一次表达式。

例如：

$$  
Z=12x_1+15x_2  
$$

或者：

$$  
2x_1+3x_2\le60  
$$

这些都是线性的。

但现实生活中，很多关系并不是一条直线。

例如：

- 广告投入越多，新增顾客增长越来越慢；
    
- 农作物产量与施肥量之间存在最佳区间；
    
- 电池充电速度会随着电量增加而下降；
    
- 车辆油耗会随着速度变化而改变。
    

这些关系往往包含平方项、乘积项、指数函数或对数函数。

一旦目标函数或约束条件不再是线性的，就需要考虑非线性规划。

# 🚩 什么时候想到非线性规划？

识别非线性规划，可以重点观察题目中的描述。

如果出现下面这些关键词，就值得提高警惕：

- 边际收益递减
    
- 最佳投入
    
- 平方关系
    
- 指数增长
    
- 对数增长
    
- 曲线变化
    
- 两个变量相乘
    
- 效率随投入变化
    

例如：

题目说：

> 广告投入越多，新增顾客增长越来越慢。

这通常意味着收益不会与预算保持线性关系。

再例如：

> 农作物产量与施肥量之间存在最佳施肥量。

继续增加肥料，产量反而可能下降。

这种现象通常可以用二次函数表示。

一个简单的判断方法是：

> **如果目标函数或约束条件已经不能写成一次表达式，而需要使用平方、乘积、指数、对数等形式，就应该考虑非线性规划。**

当然，如果只是变量必须取整数，那仍然属于整数规划，而不是非线性规划。

# 🏆 一个完整案例

一家奶茶店计划投放线上广告。

为了方便分析，老板决定把广告预算全部投放在短视频平台。

设：

- 广告投入金额为 $x$（单位：千元）。
    

根据过去几个月的数据分析，预计广告带来的新增利润可以近似表示为：

$$  
P(x)=40x-x^2  
$$

今天到底需要决定什么？

答案就是：

**广告预算投入多少。**

接下来，还要回答另一个问题：

怎样才算最好？

当然是新增利润最高。

因此，目标函数可以写成：

$$  
\max P(x)=40x-x^2  
$$

为什么不能无限增加预算？

因为老板只有 30 千元预算。

于是得到约束：

$$  
0\le x\le30  
$$

整理之后，整个模型可以写成：

$$
\begin{aligned}
\max \quad & P(x) = 40x - x^2 \\
\text{s.t.} \quad & 0 \le x \le 30
\end{aligned}
$$

这里最值得关注的是目标函数。

前面的：

$$  
40x  
$$

表示广告投入越多，利润越高。

而后面的：

$$  
-x^2  
$$

表示随着广告投入不断增加，收益增长速度开始下降。

两部分共同描述了现实中的边际收益递减现象。

可以看到，我们依然是在完成同样的建模过程：

现实问题

↓

数学表达

例如：

- 投入广告越多，收益越高 → $40x$
    
- 投入过多，广告效果下降 → $-x^2$
    

与线性规划相比，最大的变化不是建模步骤，而是数学表达更加贴近现实。

# 💻 Python 实现

由于目标函数已经不是线性的，因此不能使用 `scipy.optimize.linprog()`。

可以使用 `scipy.optimize.minimize()` 求解。

目标函数：

$$  
40x-x^2  
$$

由于 `minimize()` 默认求最小值，因此需要把目标函数取相反数：

```python
return -(40 * x - x ** 2)
```

完整代码如下：

```python
import numpy as np
from scipy.optimize import minimize

# 定义目标函数（取负号表示求最大值）
def objective(x):
    return -(40 * x - x ** 2)

# 初始值
x0 = [10]

# 决策变量范围
bounds = [(0, 30)]

# 求解
result = minimize(
    objective,
    x0,
    bounds=bounds
)

print("是否求解成功：", result.success)
print("最佳广告投入：", result.x[0], "千元")
print("最大新增利润：", -result.fun)
```

数学模型中的目标函数，在程序中直接写成一个 Python 函数。

求解器会自动寻找使目标函数最大的投入金额。

这也是非线性规划与线性规划最大的区别之一：我们不再使用矩阵描述目标函数，而是直接定义函数。

# 📌 本章小结

非线性规划并没有改变优化问题的基本思路，它只是允许模型更加真实地描述现实世界。

学习非线性规划时，可以先问自己三个问题：

1. **需要决定什么？**
    
2. **希望优化什么？**
    
3. **目标函数或约束条件还是线性的吗？**
    

如果第三个问题的答案是否定的，就应该考虑非线性规划。

从广告投放、农业生产，到能源管理、投资决策，许多现实问题都无法用一条直线描述。此时，非线性规划往往能够建立更加符合实际的数学模型。

> **现实世界很少是直线，而非线性规划正是连接现实与数学的重要桥梁。**
