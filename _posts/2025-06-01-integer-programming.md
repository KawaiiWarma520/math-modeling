## 🧠 基本概念

**一句话定义**：整数规划 = 线性规划 + 决策变量必须取整数的约束。

**三种常见类型**：

| 类型 | 变量取值 | 适用场景 |
| :--- | :--- | :--- |
| **纯整数规划** | 所有变量都是整数 | 人数、机器台数、车辆数 |
| **0-1规划** | 变量只能取 0 或 1 | 是否建厂、是否选择某方案 |
| **混合整数规划** | 部分整数 + 部分连续 | 生产数量（整数）+ 原材料用量（连续） |

## ⚠️ 核心难点

整数规划比线性规划**难求解得多**——不是因为有整数就要多算几步，而是问题的复杂度会指数级爆炸。

> 线性规划可以在多项式时间内求解，但整数规划是 **NP-hard** 问题。简单说：变量一多，计算机也可能算到天荒地老。

## 🏆 经典例子：背包问题

> 一个背包容量为 10kg，现有 4 件物品。每件物品有重量和价值，如何装使得总价值最大？

| 物品 | 重量 (kg) | 价值 (元) |
| :--- | :--- | :--- |
| 物品1 | 2 | 3 |
| 物品2 | 3 | 4 |
| 物品3 | 4 | 5 |
| 物品4 | 5 | 7 |

**模型表达**：
- 决策变量：$x_i = 1$ 表示选物品 $i$，$x_i = 0$ 表示不选
- 目标函数：$\max Z = 3x_1 + 4x_2 + 5x_3 + 7x_4$
- 约束条件：$2x_1 + 3x_2 + 4x_3 + 5x_4 \leq 10$
- 0-1约束：$x_i \in \{0, 1\}$

## 💻 代码实现

```python
import pulp

# 创建问题（注意这里是最大化）
model = pulp.LpProblem("Knapsack", pulp.LpMaximize)

# 数据
weight = [2, 3, 4, 5]
value = [3, 4, 5, 7]
n = len(weight)
capacity = 10

# 决策变量：0-1变量
x = [pulp.LpVariable(f'x{i}', cat='Binary') for i in range(n)]

# 目标函数
model += pulp.lpSum(value[i] * x[i] for i in range(n))

# 约束条件
model += pulp.lpSum(weight[i] * x[i] for i in range(n)) <= capacity

# 求解
model.solve()

print("求解状态:", pulp.LpStatus[model.status])
print("选择的物品:", [i+1 for i in range(n) if x[i].varValue == 1])
print("总价值:", pulp.value(model.objective))
```

| 模型 | 关系 |

| **0-1规划** | 整数规划的特例（0-1变量） |
| **动态规划** | 另一种解决背包问题的思路 |
