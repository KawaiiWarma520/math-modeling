# 📖 故事引入

某物流公司在优化配送路线时遇到一个问题：

他们用贪心算法得到了一条路径：

- 看起来不错
    
- 但稍微调整后可能更优
    

问题在于：

> 一旦局部调整变差，算法就不再接受它

于是算法卡住了：

> 一直停留在“还不错，但不是最优”的解

于是有人提出一个反直觉的想法：

> **有时候，允许“变差”，反而能找到更优解。**

今天要学习的，就是解决这类问题的经典方法——**模拟退火算法（SA）**。

---

# 🧠 什么是模拟退火？

一句话来说：

> **模拟退火，就是通过“逐渐降低接受坏解的概率”，从随机搜索逐步收敛到全局最优的方法。**

它来自物理过程：

> 金属退火（加热 → 冷却 → 结构稳定）

---

# 🚩 什么时候想到模拟退火？

如果题目中出现下面这些情况，就可以考虑：

- 组合优化问题
    
- 路径优化（TSP）
    
- 局部最优陷阱
    
- 解空间巨大
    
- 遗传算法不方便使用
    

例如：

- 快递路径优化
    
- 排班优化
    
- 资源分配
    
- 网络优化问题
    

一个简单判断方法：

> **如果“贪心算法容易卡在局部最优”，就考虑模拟退火。**

---

# 🧠 核心思想

模拟退火的核心是：

> 不总是接受更优解，也可能接受更差解

---

## 📌 接受概率公式

如果新解更差：

$$  
P = e^{-\frac{\Delta E}{T}}  
$$

其中：

- $\Delta E$：变差程度
    
- $T$：温度（控制随机性）
    

---

# 🧠 温度机制（核心直觉）

---

## 🔥 高温阶段

- 容易接受差解
    
- 大范围搜索
    
- 防止陷入局部最优
    

---

## ❄ 低温阶段

- 越来越保守
    
- 收敛到最优解附近
    

---

# 🧠 算法流程

```text
初始解
  ↓
随机扰动生成新解
  ↓
计算差异 ΔE
  ↓
如果更优 → 接受
如果更差 → 按概率接受
  ↓
降低温度 T
  ↓
重复迭代
```

---

# 🏆 一个完整案例

某公司要优化配送路径（TSP简化版）：

- 5个城市
    
- 目标：路径最短
    

---

## 今天要解决的问题：

> 如何避免陷入“局部最优路径”？

---

## 贪心问题：

- 每次选最近点
    
- 但整体路径可能不是最短
    

---

## 模拟退火做法：

### Step 1：随机初始路径

例如：

```text
A → C → B → E → D
```

---

### Step 2：随机交换两个点

```text
A → B → C → E → D
```

---

### Step 3：计算路径变化

- 如果更短 → 接受
    
- 如果更长 → 可能接受
    

---

### Step 4：逐步降温

- 前期大胆探索
    
- 后期逐步收敛
    

---

# 💻 Python 实现（简化版）

```python
import numpy as np
import random
import math

# 简单目标函数（越小越好）
def cost(x):
    return sum(x)

# 初始解
current = np.random.rand(5)
current_cost = cost(current)

best = current.copy()
best_cost = current_cost

T = 10  # 初始温度

for i in range(100):
    # 生成新解（扰动）
    new = current + np.random.normal(0, 0.1, 5)
    new_cost = cost(new)

    delta = new_cost - current_cost

    # 接受规则
    if delta < 0 or random.random() < math.exp(-delta / T):
        current = new
        current_cost = new_cost

        if new_cost < best_cost:
            best = new
            best_cost = new_cost

    # 降温
    T *= 0.95

print("最优解：", best)
print("最优值：", best_cost)
```

# 📌 本章小结

模拟退火解决的是：

> **如何跳出局部最优，寻找全局最优解**

学习它时，可以记住三个关键词：

1. **温度（控制随机性）**
    
2. **概率接受（允许变差）**
    
3. **逐步降温（收敛过程）**
    

---

## 🔥 和遗传算法的区别

|方法|思路|
|---|---|
|遗传算法|多个解进化|
|模拟退火|单个解随机搜索|

---

> **遗传算法靠“群体竞争”，模拟退火靠“概率跳跃”。**
